Aim of the project

The aim of the project is to create a system that can automatically create a machine learning model for a given dataset. The system will automatically select the best model for the dataset and will also compare the accuracies of models and will select the best model for the dataset.

But before training the model there are several tasks that we need to perform on the dataset.

  • Importing the dataset
  • Checking for null values
  • Checking for categorical values
  • Checking for outliers
  • Checking for skewness
  • Checking for correlation
  • Splitting the dataset into training and testing sets
  • Scaling the dataset
First we will automate these tasks then we will proceed further and complete those ones also.
